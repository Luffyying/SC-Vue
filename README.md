##  从浅入深Vue源码学习  ##


#### 说明： ####

#### 项目使用webpack、rollup进行模块化打包处理，将大文件拆分为多个独立的小文件  
####


文件目录：

dist:打包后的文件

src:主要模块代码

test:测试用例


 package.json script :

- 1.npm run build

生成vue.js文件，并监听变化

- 2.npm run dev

开启前端服务器，热更细页面

- 3.npm run buildw

打包，生成最终运行文件


#### 使用： ####
clone

npm i

npm run dev

npm run build
